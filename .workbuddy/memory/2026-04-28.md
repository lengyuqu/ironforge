# 2026-04-28

## IronForge i18n 国际化

前端 i18n 已实现（中文 + 英文）：

- `web/src/lib/i18n/index.ts` — i18n 核心（locale store + t() + formatDate）
- `web/src/lib/i18n/translations/en.json` — 英文翻译（199 keys）
- `web/src/lib/i18n/translations/zh-CN.json` — 中文翻译（199 keys，全部对齐）
- `+layout.svelte` — 初始化 locale
- `Navbar.svelte` — 语言切换器下拉菜单
- 22 个页面组件全部改造，使用 `$t('key')` 和 `formatDate()`

技术方案：Svelte 5 reactive store + JSON 文件，localStorage 持久化，自动检测浏览器语言。

后端 i18n（rg-i18n crate）待实现。
